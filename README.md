# openlibrary-go

[![Go Reference](https://pkg.go.dev/badge/github.com/raitucarp/openlibrary-go.svg)](https://pkg.go.dev/github.com/raitucarp/openlibrary-go)

A Go client library for the [OpenLibrary API](https://openlibrary.org/developers/api).
Provides fluent access to authors, works, editions, search, covers, and subjects.

## Installation

```bash
go get github.com/raitucarp/openlibrary-go
```
